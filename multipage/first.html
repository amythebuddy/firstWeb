HTML

Introduction about HTML:

  • HTML, HyperText Markup Language, is a standard language to create the basic structures for Web pages.
  • HTML was created in 1993 and it is used to now!
  • HTML is easy to learn because it is similar to English.
  • HTML uses a list of elements at the beginning of the content to tell the computer that "this is a heading", "this is an image", etc.

Basic layout in HTML:

<!DOCTYPE html>
<html>
<head>
<title>Document</title>
</head>
<body>

<h1>This is a heading </h1>
<p>This is a paragraph </p> br>
</body>
</html>

This is a heading

This is a paragraph

Explaination:

  • <!DOCTYPE html> declaration defines what kind of version the HTML is. (we are using HTML5)
  • <html> element is the root element of an HTML page
  • <head> element has meta information about the HTML page
  • <title> element is shown in the browser's title bar or in the page's tab
  • <body> element defines the document's body, and has all the visible contents, such as headings, paragraphs, images, etc.
  • <h1> element is the largest heading
  • <p> element is a paragraph

HTML Elements:

Most HTML elements have a opening and closing tag

<tagname>Type content here... </tagname>

<h1>This is a heading </h1>
<p>This is a paragraph </p>

Important: There are some elements do not have an end tag. Those are called empty elements.
Only the content inside the body section will be displayed in a browser. The title only shows in the page's tab.
