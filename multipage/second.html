CSS

Introduction about CSS:

  • CSS, Cascading Style Sheets, designs the website layout, styles, colors, and etc.
  • CSS is used with HTML to style the content of the web.
  • CSS saves a lot of time and typing, because it can add a layout to many webpages.
  • CSS can be used within HTML file, or external stylesheets, called CSS file.

Basic layout in CSS:

<!DOCTYPE html>
<html>
<head>
<title>Document</title>
</head>
<body>

<h1>This is a heading </h1>
<p>This is a paragraph </p>

</body>
</html>
